Projeto de Testes com JUnit e Mockito 📌
-

Este repositório contém um estudo prático de testes automatizados utilizando JUnit e Mockito em um projeto Java/Spring Boot.

O objetivo é explorar boas práticas de testes unitários e mocking em aplicações Spring, garantindo maior confiabilidade e manutenção do código.

🔧 Tecnologias utilizadas:
- Java 17+
-  Boot
- JUnit 5
- Mockito
- WebMvcTest
- Git para versionamento

Contexto do Projeto 🚗
-
O sistema de exemplo desenvolvido para os testes é uma locadora de carros, com funcionalidades como:

- Cadastro de veículos
- Registro de clientes
- Processamento de reservas

🎯 Objetivo

- Esse projeto foi escolhido por ser simples, mas suficiente para aplicar diferentes cenários de testes.
- Demonstrar como estruturar testes unitários em projetos Spring Boot.
- Utilizar Mockito para simular dependências e validar comportamentos isolados.
- Versionar o código no Git para controle e evolução do estudo.
